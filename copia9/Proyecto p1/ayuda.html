Guía de Uso - Sistema de Cuentas Bancarias

¿Qué hace este programa?

Este sistema permite crear y gestionar cuentas bancarias, incluyendo cuentas de ahorro y cuentas corrientes. Puedes:

  • Crear nuevas cuentas
  • Depositar y retirar dinero
  • Buscar cuentas existentes
  • Visualizar información de las cuentas
  • Búsquedas por diferentes parámetros

Cómo usarlo

  1. Ejecuta el programa desde banco.exe
  2. Sigue las instrucciones del menú en pantalla
  3. Selecciona la opción que deseas (crear cuenta, depositar, etc.)
  4. Ingresa los datos requeridos (nombre, tipo de cuenta, monto...)
  5. Asegúrate de ingresar los datos correctos

Tipos de Cuenta

  • Cuenta de Ahorro: Ideal para guardar dinero con fines de ahorro. Puedes crear las cuentas de ahorro que necesites.
  • Cuenta Corriente: Permite transacciones frecuentes, como retiros constantes. Puedes crear las cuentas corrientes que necesites, pero deben tener un depósito inicial de $500 (quinientos dólares).

Notas importantes

Los datos de las cuentas se guardan automáticamente en un archivo binario (cuentas.bin) para su persistencia.
